# 📌주요기능
- 리그 오브 레전드에 사용자 정보를 가져옴
  - 소환사 이름, 소환사 레벨, 소환사 랭크 정보, 승률, 이전 아이디, 최근 20게임에 대한 정보를 알려줌
    - 최근 게임에서는 게임 타입, 게임 종료 시간/시작 시간, 승/패, 아이템 정보, 킬/데스/어시스트, 룬/특성/스펠, CS/Gold 에 정보를 알려줌
